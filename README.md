# Sigmafetch
Minimalest sysinfo app.
# Installation
## Dependencies
- <a href="https://www.freedesktop.org/wiki/Software/wmctrl/">wmctrl</a>
## Installing Sigmafetch
- Clone the repo
```
git clone https://github.com/xjlsigmaboy/sigmafetch.git
```
- Installation
```
cd sigmafetch
chmod +x install.sh
sudo ./install.sh
```
# Supported OSs
- Arch linux, EndeavourOS
- Debian, Ubuntu
- Fedora, RHEL, CentOS
- Gentoo
- NixOS
- Void Linux
